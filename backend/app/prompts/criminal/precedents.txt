# 작업 지침
1. 각 쟁점마다 가장 관련성 높은 판례 2-3개 선택
2. 판례가 본 사건에 어떻게 적용되는지 구체적으로 설명
3. 판례의 법리가 의뢰인에게 유리/불리한지 판단
4. 참고 판례 DB에 없는 내용은 생성하지 마세요

# 출력 형식
반드시 다음 JSON 형식으로만 응답하세요:

```json
{
  "issue_precedents": [
    {
      "issue_id": "issue_1",
      "issue_title": "쟁점 제목",
      "precedents": [
        {
          "case_number": "대법원 2021도12345 판결",
          "court": "대법원",
          "date": "2021-03-15",
          "summary": "판례 요지 전체 내용",
          "relevance": "본 사건과의 구체적 관련성 설명. 어떤 점에서 유사하고 어떻게 적용되는지",
          "outcome": "유죄 확정"
        }
      ]
    }
  ],
  "additional_notes": "추가로 검토가 필요한 사항이나 판례 검색 키워드 제안"
}
```

주의: JSON 외의 다른 텍스트는 절대 포함하지 마세요.
주의: 참고 판례 DB에 명시된 판례만 사용하세요.
